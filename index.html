Welcome to the Programming Quotes API!

Programming Quotes API for open source projects.

Status: Live

Introduction

Welcome to the Programming Quotes API, an open-source project that provides a vast collection of programming quotes and their authors. This API allows you to explore, search, and enjoy programming-related wisdom from various sources.

Getting Started

To get started, you can use the following endpoints:

  • GET /quotes Retrieve all quotes.
  • GET /quotes/random Get a random quote.
  • GET /quotes/{id} Get a quote by ID.
  • GET /quotes/author/{authorName} Get quotes by a specific author.

Author Info

For additional information about authors, you can use the Wikipedia API:

  • To get an author's image: https://en.wikipedia.org/w/api.php?action=query&titles=Author%20Name&prop=pageimages&format=json&pithumbsize=250
  • To get author info: https://en.wikipedia.org/w/api.php?action=query&titles=Author%20Name&prop=extracts&format=json&exintro=1

Replace "Author%20Name" with the actual name of the author.
